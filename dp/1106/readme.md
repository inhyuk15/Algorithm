## 문제 정의
**dp[n]
</br>
= n명에 드는 비용
</br>
= [k 번 도시에 k''명을 얻기 위한 최소 단위 k'원을 제외한

n-k''명에 드는 비용] + k'들의 최소값
</br>
즉 [n] = min([n-k] + k'')로 정의됨**
</br>
base case는 각 도시마다 최소 단위 k''가 된다.


a[i][0] = 인원 당 비용, a[i][1] = 인원
</br>
c = 필요한 인원, n = 도시의 수
</br></br>
pseudo code
```cpp:dp.cpp
for i <- 0 1 to c
	for j <- 1 to n
		if(i - a[j][0] >= 0)
			dp[i] = min(dp[i], dp[i-a[j][0]] +a[j][1])
	이전에 구했던 최소비용으로 최대 인원을 구하는 값 갱신하기
```
